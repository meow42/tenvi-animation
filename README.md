# 冲锋岛角色动画

## 项目简介
提取冲锋岛（Tenvi）原作素材制作的骨骼动画，用于同人作品创作。  
使用龙骨（DragonBones）制作，仓库中仅存放龙骨工程文件，需自行用龙骨导出指定格式文件。

> 龙骨版本：DragonBones Pro 5.6.3  
> 支持导出序列帧、Spine格式数据

## 制作计划
- [x] 机甲基本动作（站、走、跑、跳、被击、上Buff、空手普攻）
- [ ] 机甲进阶动作（冲锋、践踏、下蹲、飞行等）
- [ ] 机甲武器动作（单/双手的剑/斧/锤/枪械、副手盾牌）
- [ ] 机甲部件换装（舱盖、底盘、手部、腿部、推进器、主/副手武器）
- [ ] 冒险者（小人儿）基本动作
- [ ] 冒险者装备
- [ ] 龙人族守护者：幼年魔龙
- [ ] 精灵族守护者：化身/阿凡达
- [ ] 各类序列帧NPC
